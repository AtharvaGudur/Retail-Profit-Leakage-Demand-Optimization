## 📌 Project Overview

In modern retail environments, organizations often experience **declining profitability despite strong sales performance**. This is typically caused by hidden inefficiencies such as excessive discounting, high return rates, stockouts, vendor delays, and poor demand forecasting.

This project addresses these challenges by building a **comprehensive analytics solution** that integrates multiple data sources and transforms raw data into **actionable business insights**.

The solution provides:

* A **centralized data model**
* Advanced KPI calculations
* Interactive Power BI dashboards
* Clear visibility into **financial and operational inefficiencies**

---

## 🎯 Business Problem Statement

The retail business faced the following critical issues:

* 📉 Declining **net profit despite increasing sales**
* 📦 Frequent **stockouts in high-demand products**
* 🔁 High **returns and refund rates**
* 💸 Overuse of **discounts reducing margins**
* 🚚 **Vendor delays** affecting product availability
* 📊 Lack of **real-time visibility into performance metrics**

This project aims to solve these problems by delivering a **data-driven decision support system**.

## 📈 Key Metrics & Calculations

* **Gross Margin %** = (Revenue - Cost) / Revenue
* **Net Profit** = Revenue - Cost - Discounts - Returns
* **Stockout Loss** = Estimated lost sales when inventory = 0
* **Fill Rate** = Supplied Quantity / Ordered Quantity
* **Promo ROI** = Incremental Profit / Promotion Cost
* **Return Rate** = Returned Orders / Total Orders
* **Vendor SLA Breach %** = Delayed Deliveries / Total Deliveries
